Google Sheets API Connection

📋 How to get your Apps Script URL:
  1. Open your Google Sheet → Extensions → Apps Script
  2. Paste the google-apps-script.gs file contents
  3. Run SETUP_EVERYTHING once (creates all sheets + seeds products)
  4. Click Deploy → New Deployment
  5. Type: Web App
  6. Execute as: Me
  7. Who has access: Anyone ← very important
  8. Click Deploy → Authorize → Copy the URL
  9. Paste the URL below and click Save, then Test
URL must end with /exec
⚠️ If you see "Failed to fetch":
The most common cause is the deployment setting "Who has access" is not set to Anyone. Go back to Apps Script → Deploy → Manage deployments → Edit → change access → re-deploy → copy the new URL.
