Google Apps Script Setup Guide

  1. Go to Google Sheets and create a new spreadsheet.
  2. Add these sheets: Orders (columns: order_id, name, phone, address, items, total, status, date) and Products.
  3. Open Extensions > Apps Script and paste the google-apps-script.gs code.
  4. Deploy as Web App — set access to Anyone. Copy the URL.
  5. Paste the URL above and click Save.
