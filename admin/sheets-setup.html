Setup Instructions

1

Open Google Apps Script

Go to script.google.com, create a new project, and paste the contents of apps-script.gs.

2

Deploy as a Web App

Click Deploy → New Deployment → Web App. Set Execute as: Me · Who has access: Anyone. Click Deploy.

3

Paste the URL above

Copy the URL ending in /exec, paste it in the field above, and click Save & Test.

4

Done!

New orders will sync to a sheet named Orders, which is created automatically.
